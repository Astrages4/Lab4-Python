## Лабораторная работа №4. Симуляция с пользовательскими коллекциями и псевдослучайной моделью
### Вариант 1. "Библиотека"

### Основные возможности:
        - Управление коллекцией книг через пользовательские коллекции
        - Индексация книг по ISBN, автору, году издания и жанру
        - Псевдослучайная симуляция библиотечных операций
        - Поддержка выдачи и возврата книг
        - Статистика работы библиотеки

#### Две пользовательские коллекции:
    - BookCollection (списочная через композицию)
    - IndexDict (словарная для индексации)

#### Коллекции поддерживают:
    - __iter__ — итерирование
    - __len__ — длину
    - __getitem__ — доступ по индексу/ключу
    - Поддержку срезов (для списочной коллекции)
    - Добавление и удаление элементов

#### В словарной коллекции реализована логика индексации книг

#### Также реализованы:
    - Один базовый класс Library
    - Два производных класса: EnhancedLibrary и StatisticsLibrary

#### Магические методы:
    - __call__ для поиска книг
    - __repr__ для строкового представления
    - __contains__ для проверки наличия элементов

#### Есть функция симуляции run_simulation()


### Классы
### Модели данных
#### 1. Book (models/book.py)
##### Класс, представляющий книгу в библиотеке.

##### Поля:
    - title — название книги
    - author — автор
    - year — год издания
    - genre — жанр
    - isbn — уникальный идентификатор

##### Магические методы:
    - __repr__() — строковое представление
    - __eq__() — сравнение по ISBN
    - __hash__() — для использования в словарях

##### Пользовательские коллекции
#### 2. BookCollection (models/collections.py)
##### Списочная коллекция книг, реализованная через композицию.

##### Реализованные методы:
    - __getitem__() — доступ по индексу и срезы
    - __len__() — количество книг
    - __iter__() — итерирование
    - __contains__() — проверка наличия книги
    - add() — добавление книги
    - remove() — удаление книги

##### Методы поиска по автору, жанру, году

#### 3. IndexDict (models/collections.py)
##### Словарная коллекция для индексации книг.

##### Индексы:
    - По ISBN (быстрый доступ)
    - По автору (группировка)
    - По году издания
    - По жанру

##### Реализованные методы:
    - __getitem__() — доступ по ключу (ISBN или кортежу)
    - add_book() — добавление во все индексы
    - remove_book() — удаление из всех индексов
    - get_stats() — статистика индексов

### Классы библиотек
#### 4. Library (core/library.py)
##### Базовый класс библиотеки.

##### Функциональность:
    - Управление коллекцией книг
    - Индексация через IndexDict
    - Поиск книг по различным критериям
    - Магический метод __call__() для поиска

#### 5. EnhancedLibrary (core/library.py)
##### Расширенный класс библиотеки с системой выдачи книг.

##### Дополнительные возможности:
        - Выдача книг читателям
        - Возврат книг
        - Отслеживание выданных книг
        - Получение списка доступных книг

#### 6. StatisticsLibrary (core/library.py)
##### Библиотека со статистикой операций.

##### Дополнительные возможности:
        - Подсчет всех операций (добавление, удаление, поиск)
        - Статистика использования библиотеки

### Симуляция
#### 7. LibrarySimulation (core/simulations.py)
#####Класс для управления псевдослучайной симуляцией.

##### События симуляции:
        - Добавление новой книги (25%)
        - Удаление случайной книги (15%)
        - Поиск по автору (15%)
        - Поиск по жанру (15%)
        - Обновление индекса (5%)
        - Попытка получить несуществующую книгу (5%)
        - Выдача книги (10%)
        - Возврат книги (5%)
        - Поиск по году (5%)

#### 8. Функция run_simulation() (core/simulations.py)
##### Основная функция для запуска симуляции.

##### Параметры:
        - steps — количество шагов симуляции
        - seed — seed для генератора случайных чисел (обеспечивает воспроизводимость)
